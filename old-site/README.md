# upper.io docs

## How to install and run

Install [luminos](menteslibres.net/luminos):

```
make deps
```

Run the server:

```
make run
```

And open 127.0.0.1:9000.

## Directory layout

Some important files and directories:

```sh
# Main page template
sites/default/templates/index.tpl

# Web root
sites/default/webroot

# Markdown files
sites/default/content
```
